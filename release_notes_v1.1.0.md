# Release Notes v1.1.0

**DARK-ART v1.1.0** is now available. This update brings enhanced data visibility, improved export options, and a refined user interface to streamline your analysis workflow.

## New Features

### **Data Management & Export**
- **Datasheet Viewer**: A new dedicated datasheet viewer allows you to inspect your raw dataset directly within the app.
- **Improved Export Capabilities**:
  - **Datasheet**: Added "Copy to Clipboard", "CSV", and "Excel" export buttons for quick data extraction.
  - **Result Table**: Added "Copy to Clipboard", "CSV", and "Excel" export buttons for quick table export.

### **User Interface (UI)**
- **Refined Visualisation Layout**: Improved consistency and spacing across the Visualization and Result Table tabs to prevent layout shifts.
- **Brand Identity**: Added the official application **logo** to the header and browser tab icon for better recognition.
- **Mobile Friendly**: Optimized the layout for improved responsiveness on smaller screens and mobile devices.

## Bug Fixes
- **Stability**: Fixed a critical error (*"max not meaningful for factors"*) that prevented specific categorical analyses from running correctly.
